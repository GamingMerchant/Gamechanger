Game Image Generator

Click the buttons below to generate preview images for your games.

Instructions:

  1. Click on a game button to generate its preview image
  2. Click "Download Current Image" to save the image
  3. Upload the downloaded images to your GitHub repository in the assets/images folder
  4. Make sure to name them: snake-game.png, pong-game.png, tetris-game.png, and space-invaders-game.png
